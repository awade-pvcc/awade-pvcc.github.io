OPERATING SYSTEM TERMINOLOGY

Term Definition
operating system The software that controls the overall operation of the computer including file management, the user interface, and the program execution environment.
job queue A storage where jobs wait to be executed, especially in a batch processing system.
FIFO Stands for: First In First Out. This is the order jobs are executed when waiting in a queue.
interactive processing A feature that allows a running program to have communication with the user via remote terminals.
time-sharing A feature in operating systems that offer services to multiple users at the same time.
multiprogramming A time-sharing method that divides the time into intervals and each job is restricted to one interval at a time.
application software Comprised of the programs needed to perform the specific tasks dedicated to the device's utilization.
system software Performs tasks that are common to computer systems and provides the foundation the application software requires.
utility software Software units that generally consist of programs that peform activities essential to computer installation but not included in the operating system.
graphical user interface (GUI) A representation of files and programs as icons on a display that interact with the user and can be manipulated by the user.
shell An older style of interface where communication with the user is performed through textual messages with a keyboard and monitor.
kernel The internal part of an operating system that contains software components that execute the most basic functions needed by the computer installation.
device driver Software units that are a component of the kernel and they communicate with the controllers to execute operations on devices attached to the machine.
file manager A unit of the kernel that performs a basic function such as the coordination of the use of mass storarge.
memory manager A component of the operating system's kernel and tasked with coordinating the use of main memory in the machine.
virtual memory The large space created with the technique of 'paging' to store more data than the machine has space to store.
boot strapping The procedure that transfers the operating system from mass storage and into main memory when the machine is turned on.
ROM Non-volatile memory that can be read but not altered, and stand for Read-Only Memory.
deadlock A resource condition where multiple processes are blocked from progressing because the needed resources are in use by another process.
spooling A technique of holding data for output at a later time because it is more convenient, such as when a resource is available.
super user (or administrator) A user of the device that gains privileged access to the operating system. This elevated access allows an install or upgrade of software, make changes to the operating system, grant elevated privileges to other users, and perform maintenance other users are not able to perform.
multi-factor authentication An authentication method that requires two or more verification factors to gain access to a device or application.
