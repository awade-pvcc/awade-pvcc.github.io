Networking and The Internet

Category Terms Definition
Network Communications
  • LAN
  • MAN
  • WAN
  • Stands for Local Area Network. Used in a collection of computers in a single building or complex.
  • Stands for Metropolitan Area Network. Network of intermediate size, like one covering a local community.
  • Stands for Wide Area Network. This type connects machines over a greater distance, like in bordering cities or cities on opposite sides of the world.
Network models
  • bus network
  • star network
  • Links are created from each computer to a central location where they are connected to a hub.
  • Machines are connected over a intermediary central machine.
Protocols
  • CSMA/CD Ethernet
  • CSMA/CA Wifi
  • Stands for Carrier Sense, Multiple Access with Collision Detection. This protocol controls the right to transmit messages over the network and authorizes the broadcast of each message to all the machines on the bus.
  • Stands for Carrier Sense, Multiple Access with Collision Avoidance. A policy that is used to avoid the collision of wireless network traffic between machines.
Combining networks
  • repeater
  • bridge
  • switch
  • router
  • gateway
  • A device that passes signals back and forth between the two original buses without the meaning of the signals being considered.
  • Device that connects two buses but does not pass all messages over the connection. The destination address is used to forward a message when that computer is on the other side. Two machines on the same side of the bridge can message each other without interfering with communication.
  • Allows connection with two or more buses.
  • A device that handles the connection between networks to form an internet.
  • The "point" a network is linked to the internet and is a passageway between the network and outside world.
Client/Server model
  • P2P: peer-to-peer
  • distributed system
  • cloud computing
  • A model that provides service to and receive service from each other.
  • Consist of software units that execute as processes on different computers.
  • Used in distributed systems that utilize large pools of shared computer on the network that can be distributed to clients as needed.
Internet Addressing
  • Tier 1
  • Tier 2
  • Tier 3
  • The top of the hierarchy of ISPs. They have high-speed, high-capacity, and international WANs. Usually operated by large companies in the communications business.
  • These ISPs are more regional in scope and less potent in their capablities.
  • ISPs that are sometimes called access ISPs and is an independent Internet, or instranet, that is operated by a single authority that is in the business of supplying Internet access to individual homes and businesses.
IP address
  • ICANN
  • DNS
  • TLD: Top-level domain
  • Country-code TLDs
  • Stands for the Internet Corporation for Assigned Names and Numbers. Numbered IP addresses are given by this non-profit corporation which coordinates the Internet's operation. Link also provided.
  • A collective of name servers that are an Internet-wide directory system that uses the name servers to provide address translation services to clients.
  • The suffix in a url that reflects the domain classification.
  • Two-letter TLDs that are specfic to each country around the world.
Internet protocols
  • FTP
  • HTTP
  • SMTP
  • Stands for File Transfer Protocol. This is an application that allows access of another computer across great distances.
  • Stands for Hyper Text Transfer Protocol. Allows for more sophisticated webservers and browsers to be accessed through this more powerful application.
  • Stands for Simple Mail Transfer Protocol. Allows for two computers on the network to interact when sending an email message from one host to another.
World Wide Web
  • URL
  • client-side activities
  • server-side activities
  • Stands for Uniform Resource Locator. A unique address that is used to locate and retrieve doucments on the Web.
  • When input is needed by the browser through form submissions or if the webpage involves animation.
  • When input is needed by the webserver through form submissions or if the webpage involves animation.
Internet four-layer software
  • application layer
  • transport layer
  • network layer
  • link layer
  • The first of the layers for Internet software. The message or original communication usally begins here and consists of software units that use Internet communication to carry out their tasks.
  • The second layer for Internet software and is used to carry or transfer the communication with the proper format.
  • The third layer of Internet software and is tasked with deciding the direction the packet should be sent at each step along its path through the Internet
  • Fourth and final layer of the Internet software and is responsible for transferring the packet.
Internet is:
  • packet-switching network
  • fault tolerant
  • redundant
  • A network that sends digital data by breaking the data into blocks of bits called packets that contain the data and control information for routing the data.
  • A process that allows a response to a failure in software or hardware through the operating system.
  • The process of routing data between two points in more than one way to increase the reliablility of the Internet to more devices and users.
Cybersecurity
  • malware
  • virus
  • worm
  • Trojan horse
  • spyware
  • phishing
  • DDoS attack
  • Malicious software that attacks a computer system or network in an effort to disrupt, cripple, or cyphen off data and information.
  • Software that infects a computer by inserting itself into programs that already reside on the machine.
  • Self-driving program that transfers inself through a network while residing in the computers and sending copies of itself to other computers.
  • A program that enters a system disguised as a wanted program that is willingly imported by a user.
  • Software that collects information on activites of the system it is on and reports the information back to the originator of the attack.
  • A technique that request information by asking for it by sending numerous request through emails. Each email acts as bait and is used to gain access to a system.
  • Stands for Distributed Denial of Service Attack. This attack involves a large number of messages that bombard the server from software that is planted on multiple computers.
Protection
  • firewall
  • spam filter
  • proxy server
  • antivirus software
  • Cryptography
  • public key encryption
  • A technique that filters network traffic through a point in the network to prevent any harm to the system.
  • Firewalls designed to block unwanted emails.
  • A software unit that is an intermediary between a client and server that shields the client from adverse actions of the server.
  • Used to detect and remove the presence of found viruses and malware.
  • The study of sending and receiving messages securely in the presence of adversaries.
  • A technique in the field of Cryptography by which the encryption and decryption algorithm is known, one of the keys is known, and the message is secure.
