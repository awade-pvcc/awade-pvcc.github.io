Programming Languages

Term Definition
machine language A system where programs that are used with modern computers are encoded with numeric digits.
debugging Locating and correcting errors within programs.
assembler A program that converts mnemonic expressions into machine language instructions.
assembly language A collective representation of programs through a mnemonic system.
translator A program that converts high-level primitives into machine-language programs.
compiler A program that translated several machine instructions into short sequences to simulate the activity requested by a single high-level primitive.
interpreter A program that executes the instructions as they are translated from the higher level form instead of recording the translated version for later use.
cross-platform software Software that is independent of an operating system's design and the machine's hardware design so that it is executable throughout a network.
natural languages Languages such as German, Latin, and English, that evolved over time without formal grammatical analysis.
formal languages Programming languages that are precisely defined by grammar analysis.
machine independence Programming languages with higher level primitives that do not rely on specific characteristics of a particular machine software.
high-level programming language Provide more abstractions for the programmer and allow easier reading and writing of a program.
object-oriented programming (OOP) Software system with a collection of units called object where each is able to perform actions that are immediately related to itself as well as requesting action of other objects.
scripted language A subset of imperative programming languages that are used to perform administrative tasks instead of developing complex programs.
data types:
  • integer: Numeric data consisting of whole numbers that are stored using two's complement notation.
  • float: Numeric data that might contain values other than whole numbers and probably stored in floating-point notation.
  • string: A combination of characters that are constant or variable.
primitive data type Included as primiitives in programming languages are int for integer and char for character.
data structure The arrangment of the data, such as the variables, in a program.
Python collection data structures:
  • list: Used to store multiple items in a single variable and are ordered, changeable, and allow duplicate values.
  • tuple: Used to store multiple items in a single variable and are ordered, unchangeable, and allow duplicate values.
  • set: Used to store muliple items in single variable and are unordered, unchangeable, and unindexed.
  • dictionary: Used to store values key pairs and are ordered, changeable, and do not allow duplicates.
function Set of instructions for performing a task that can be used as an abstract tool by other program units.
local variable A variable declared within a function and can only be referenced within that function.
global variable Variables that are not restricted to a particular part of a program.
source program The original form of a program.
object program The translated version of a program.
